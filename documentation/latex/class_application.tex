\doxysubsection{Detailed Description}
Manages the \doxylink{class_packer}{Packer} application, providing a command-\/line interface. 

The \doxylink{class_application}{Application} class acts as the user interface for the \doxylink{class_packer}{Packer} application. It allows users to interact with \doxylink{class_packer}{Packer} through various commands, such as setting paths, configuring options, and executing packing operations. Users can provide input and receive output through the console.
