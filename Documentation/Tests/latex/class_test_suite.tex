\doxysubsection{Detailed Description}
A simple test suite framework for writing and running tests. 

This class provides a basic framework for creating and running test cases. Test cases are added to the suite using the {\ttfamily add\+\_\+test} method, and then the {\ttfamily run\+\_\+tests} method is called to execute all the test cases.
