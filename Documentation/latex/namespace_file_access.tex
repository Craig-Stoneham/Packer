\doxysubsection{Detailed Description}
A namespace containing utilities for file access operations. 

The {\ttfamily \doxylink{namespace_file_access}{File\+Access}} namespace provides functions and a callback mechanism for performing various file access operations such as reading, writing, copying, moving, and removal of files.
