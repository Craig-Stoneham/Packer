\doxysubsection{Detailed Description}
A utility class for packing and managing files based on user-\/defined rules. 

The {\ttfamily \doxylink{class_packer}{Packer}} class allows you to specify rules for packing and managing files in a directory. It supports various packing modes, extension handling, file suffixes, and more.
